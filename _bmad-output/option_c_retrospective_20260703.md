# Option C Retrospective — H-C1 verdict input

## VERDICT: H-C1 REJECTED — Option C (policy-shock size throttle) is DEAD

**H-C1 stated:** flagged high-policy-risk days had materially worse realized PF for our systems than unflagged days. **The data says no — and the point estimates lean the opposite way.**

- **YANK sealed backtest (N=78, the only statistically meaningful sample):** flagged days PF 2.529 vs unflagged 1.651 (DAY variant); WINDOW variant 2.317 vs 1.519. Permutation p = 0.44/0.59 for "flagged worse" — no evidence of harm; expectancy per trade was *higher* on shock days ($141 vs $115). Coherent mechanism: YANK is a bearish-only volatility-harvesting system with a 6:1 TP:SL structure; shock regimes feed it.
- **MIM-NB live (N=7):** all three flagged-day trades won (+$1,048); all four losses came on unflagged days. Anti-H-C1.
- **s26 paper collector (N=49):** essentially all profit on flagged days (PF 5.11 flagged vs 1.02 unflagged). Anti-H-C1.
- **s27 paper (N=32):** the only sample in the hypothesized direction (flagged PF 0.12 vs 1.13), p = 0.135, N_flagged = 5 — not significant.
- **GAP-1 / YANK live:** too few trades for any inference.

**Decision:** the defensive throttle would have *cost* money on every deliverable system. Option C dies here, cheaply, exactly as the strategy doc's kill-condition intended. **Explicit warning:** the flagged-days-are-BETTER pattern is a new, untested, post-hoc observation. Turning it into an "upsize on shock days" rule without a fresh pre-registration would be the iteration-loop failure pattern verbatim. If pursued at all, it belongs inside the already-approved Option B prereg (shock-aftermath scout), not as a bolt-on.

**Method integrity notes:** the flag calendar (70 dates, 12 windows; `data/macro/policy_shock_calendar_2025_2026.csv`) was compiled from the public news record by a research agent with no access to our P&L, and frozen before this analysis script was written. Flag coverage is high (24% of trading days DAY-flagged, 44% WINDOW-flagged) because H2-2025→H1-2026 genuinely was wall-to-wall policy shock; that dilutes contrast and is itself a finding — a throttle that stands down 44% of the time is barely a strategy at all. YANK live rows exclude pre-idempotency replay contamination (symbol IS NULL rows from the 06-12 backfill era). Live samples are tiny; only the YANK backtest sample supports inference.
